# Todo List in Python 📝

Un semplice programma da terminale per gestire una lista di attività (ToDo).  
Permette di aggiungere, completare ed eliminare task, con salvataggio automatico in JSON.

## 🚀 Come usarlo
```bash
python todo.py

- Aggiungere nuovi task
- Visualizzare la lista
- Segnare task completati
- Eliminare task
- Salvataggio automatico
